## Game-Day down-distance matrix

### Down-and-Distance Decision Matrix

**1ST & 10 Call Distribution:**

**Run (45% of calls):**
- Inside Zone (with RPO tags available)
- Outside Zone
- Gap schemes (Power, Counter, Duo)
- From formations: Vary between 11, 12, 21 personnel

**Pass (40% of calls):**
- Play-action shots (max protection with back/TE help)
- Quick game with motion to identify coverage
- Intermediate routes (levels, smash, sail)
- From formations: Primarily 11, mix in 12

**RPO (15% of calls):**
- Slant RPO
- Bubble RPO
- Stick RPO
- From formations: 11 personnel spread looks

**Decision Factors:**
- Field position: Own territory (lean run), opponent territory (more aggressive)
- Score: Leading (more runs), trailing (more passes)
- Personnel matchup: Advantage (exploit it), disadvantage (create better look)
- Defensive tendency: Run defense (pass), pass defense (run), balanced (mix)
- Down history: If consistently stuffed on 1st down, change approach

**2ND & SHORT (1-3 yards)**

**Pass (50% of calls):**
- Play-action deep shots
- Intermediate routes off PA
- Boot concepts
- Empty formations to spread defense

**Run (30% of calls):**
- QB power/designed runs
- Power schemes with lead blockers
- Outside zone

**RPO (20% of calls):**
- Run-heavy RPOs
- Glance RPO
- Slant RPO

**Decision Logic:**
- Defense expecting run → pass
- Defense playing soft → run to set up manageable 3rd down
- Best receiver has single coverage → take shot
- QB mobile → designed run option
- Large offensive linemen advantage → power run

**2ND & MEDIUM (4-6 yards)**

**RPO (40% of calls):**
- Multiple RPO concepts based on box count
- Bubble, slant, stick, hitch variations

**Pass (35% of calls):**
- Quick game (mesh, drive, snag)
- Intermediate routes
- Screens to backs

**Run (25% of calls):**
- Misdirection (counter, trap)
- Draw plays
- Zone runs

**Decision Logic:**
- Defensive box count dictates RPO decision
- If defense shows blitz → screen or hot route
- If defense plays zone → work underneath
- If defense loads box → throw
- If defense plays light box → run

**2ND & LONG (7-9 yards)**

**Pass (50% of calls):**
- Intermediate crossing routes
- Levels concepts
- Sail/flood concepts

**Screen (30% of calls):**
- RB screens (slip, middle, swing)
- WR screens (tunnel, bubble)
- TE screens

**Run (20% of calls):**
- Draw plays
- Inside zone
- Counter schemes

**Decision Logic:**
- Defense expecting pass → screen or draw
- Defense playing zone → find holes in coverage
- Defense in man → crossing routes and picks
- Defense bringing pressure → screen game
- Setting up 3rd & manageable → safe underneath throw or draw

**2ND & VERY LONG (10+ yards)**

**Screen (45% of calls):**
- All screen variations
- Emphasis on explosive potential

**Run (30% of calls):**
- Draw plays
- Inside zone with cut-back
- Sweep if defense is pinning ears back

**Pass (25% of calls):**
- Deep shots if single-high
- Intermediate if defense playing soft
- Conservative if just need some yards

**Decision Logic:**
- Make 3rd down manageable
- Take shot if defense gives it
- Don't take sack
- Get some yards back

**3RD & SHORT (1-2 yards)**

**Run (50% of calls):**
- QB sneak with both backs pushing
- Power with lead back
- Inside zone
- Duo

**Play-Action (30% of calls):**
- Boot concepts
- PA shots off run action
- Waggle

**Quick Pass (20% of calls):**
- Slants
- Quick outs
- RPO concepts

**Decision Logic:**
- If defense loads box → play-action or quick pass
- If can get it with sneak → sneak
- If offense executing physically → power run
- If defense respecting run → easy completion with PA

**3RD & MEDIUM (3-5 yards)**

**Pass (50% of calls):**
- Quick game with pick/rub concepts
- Mesh/crossing routes
- Drive concept
- Snag concept

**Designed QB Run (25% of calls):**
- QB power
- QB draw
- Read option

**RPO (25% of calls):**
- Slant RPO
- Stick RPO
- Quick game RPOs

**Decision Logic:**
- If man coverage → rubs and picks
- If zone → sit in holes
- If mobile QB and light box → QB run
- If blitz coming → hot route or screen

**3RD & MEDIUM-LONG (6-9 yards)**

**Pass (70% of calls):**
- Mesh/crossing concepts
- Levels
- Smash
- Sail
- Man-beater routes (option, choice, pivot)

**Screen (20% of calls):**
- Screens to backs
- If pressure expected

**Conservative Pass (10% of calls):**
- Routes to sticks
- Checkdowns
- If just need first down

**Decision Logic:**
- Coverage type dictates route concept
- If man → man-beaters and option routes
- If zone → flood zones and sit in holes
- If pressure → screen or hot
- If need first down only → throw to sticks

**3RD & LONG (10+ yards)**

**Pass (60% of calls):**
- Max protection shots (7-8 blockers)
- Deep crossing routes
- Vertical routes
- Flood concepts

**Screen (25% of calls):**
- All screen variations
- Hope for missed tackles

**Conservative (15% of calls):**
- Routes to sticks
- Hope for YAC
- Avoid sack, set up punt

**Decision Logic:**
- If max protection holds → take shot
- If pressure coming → screen
- If just need some yards → work underneath
- Avoid negative play
- Field position matters for punt

**4TH DOWN**

**Decision Tree:**
1. Field Position
   - Opponent territory beyond FG range → Go for it
   - Opponent territory in FG range → Attempt FG unless score demands TD
   - Midfield → Depends on yards needed and score
   - Own territory → Punt unless desperate

2. Yards Needed
   - 4th & 1-2: High confidence in conversion
   - 4th & 3-5: Depends on offense execution and score
   - 4th & 6+: Usually punt unless trailing significantly

3. Game Situation
   - Trailing in 4th quarter → More aggressive
   - Tied/close game → Calculated risk
   - Leading → Usually conservative unless early in game

**If Going For It:**
- 4th & 1-2: QB sneak, power run, or quick pass
- 4th & 3-5: Quick passing concept or QB run
- 4th & 6+: Shot play or intermediate route
