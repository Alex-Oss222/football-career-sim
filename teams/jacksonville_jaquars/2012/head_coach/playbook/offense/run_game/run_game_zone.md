### Defensive Front & Coverage Response Matrix

**vs. 4-3 Base Defense:**
- **Run:** Inside zone to 3-tech, outside zone to edge
- **Pass:** Attack MLB in coverage
- **RPO:** Reads to bubble are good

**vs. 3-4 Base Defense:**
- **Run:** Outside zone, counter schemes
- **Pass:** Attack OLBs in coverage
- **RPO:** Scan box count carefully

**vs. Nickel Defense (4-2-5):**
- **Run:** More difficult, use misdirection
- **Pass:** Attack nickel back
- **RPO:** Great for RPOs, count box

**vs. Dime Defense (4-1-6):**
- **Run:** Attack light box with runs
- **Pass:** All pass game available
- **RPO:** Box should be light

**vs. Bear Front (0-technique):**
- **Run:** Trap, wham schemes
- **Pass:** Harder to protect inside
- **RPO:** Watch for loaded box

**vs. Over Front:**
- **Run:** Pin pull, duo away from 3-tech
- **Pass:** Normal
- **RPO:** Normal

**vs. Under Front:**
- **Run:** GT counter, ISO to weak side
- **Pass:** Normal
- **RPO:** Normal

**vs. Tite Front:**
- **Run:** Sweep, toss to edges
- **Pass:** Protected inside
- **RPO:** Normal

**vs. Wide Front:**
- **Run:** Trap, duo inside
- **Pass:** Normal
- **RPO:** Normal
