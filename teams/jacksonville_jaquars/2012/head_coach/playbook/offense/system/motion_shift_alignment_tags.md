## Source Extract: Motion Usage & Pre-Snap Adjustments

### Motion Usage & Pre-Snap Adjustments

**Types of Motion:**

**Jet Motion:**
- WR runs full speed across formation
- **Purpose:**
  - Identify man vs. zone (does DB follow?)
  - Create numbers advantage (move defender)
  - Fake jet sweep
  - Motion into better alignment
- **When to use:** Every formation, all situations

**Orbit Motion:**
- WR or RB arcs from one side to other
- **Purpose:**
  - Same as jet but slower
  - Better for reading defensive adjustments
  - Can fake orbit sweep
  - Can motion into backfield

**Return/Shuffle Motion:**
- Player motions across then returns to original spot
- **Purpose:**
  - Pure coverage identification
  - Force defensive communication
  - See how defense handles motion
  - No handoff threat

**TE Trade:**
- TE motions from one side of formation to other
- **Purpose:**
  - Change formation strength
  - Force front adjustment
  - Identify where 3-tech moves
  - Create different run gaps

**RB Motion:**
- RB motions from backfield to slot
- **Purpose:**
  - Create passing matchup (LB on RB in space)
  - Empty then motion back (force communication)
  - Identify box count changes
  - Protection adjustments

**Pre-Snap Shifts:**
- Multiple players change positions
- **Purpose:**
  - Force defensive communication
  - Create confusion
  - Get into better formation
  - Time snap for advantage

**Motion Timing:**
- **Early Motion (12-15 seconds):** Get into formation
- **Medium Motion (8-10 seconds):** Force adjustments
- **Late Motion (5-7 seconds):** Coverage identification
- **Snap-at-motion:** Jet sweep threat, defensive confusion

**Reading Defensive Response:**
- **No one moves:** Zone coverage likely
- **One player follows:** Man coverage confirmed
- **Multiple adjust:** Zone with some man principles
- **Front adjusts:** Note where 3-tech goes, where LBs fit
- **Safeties rotate:** Note rotation direction and timing
- **Late rotation:** They're disguising, read post-snap

**Using Motion Every Play:**
- Use motion 85-90% of snaps in Q1 for information
- Continue using 70-80% throughout game
- Forces defense to communicate every play
- Creates mistakes when defense tired
- Provides consistent coverage information

## Source Extract: Formation Selection Process

### Formation Selection Process

**Formation Selection Criteria:**

1. **Personnel Package:** Already determined by down-distance-situation
2. **Field Position/Hash:** Affects space available
3. **Coverage Expected:** Match formation to exploit coverage
4. **Pressure Expected:** Need protection built in?
5. **Play Type:** Run, pass, RPO, screen?
6. **Matchup:** Create specific matchup wanted?

**Formation Principles:**

**Compressed Formations (Bunch, Stack, Tight):**
- **When:** 
  - Red zone
  - Need rub routes vs. man
  - Want to create space via bunch
- **Advantage:** Natural picks, hard to defend
- **Disadvantage:** Less horizontal stretch

**Spread Formations (Trips, Empty, 3x1):**
- **When:**
  - Need to throw
  - Want horizontal stretch
  - Force defense to spread
- **Advantage:** Create space, expose matchups
- **Disadvantage:** Protection can be vulnerable

**Balanced Formations (2x2, Pistol):**
- **When:**
  - Want to run or pass
  - Don't want to tip hand
  - Standard situations
- **Advantage:** Balanced threat
- **Disadvantage:** No inherent advantage

**Heavy Formations (12, 13, 21, 22 personnel):**
- **When:**
  - Want to run
  - Short yardage
  - Goal line
- **Advantage:** Physical advantage, can still PA
- **Disadvantage:** Defense knows run more likely

**Empty Formations (00 personnel, Empty sets):**
- **When:**
  - Must pass
  - Want max route combinations
  - Force light box
- **Advantage:** Matchups in space, 5 routes
- **Disadvantage:** QB must read and deliver

**Formations to Attack Specific Coverages:**
- **vs. Cover 1:** Bunch/stack for picks, spread for isolations
- **vs. Cover 2:** Trips to stretch, verticals
- **vs. Cover 3:** 2x2 for levels/floods, trips for seam/wheel
- **vs. Cover 4:** Compressed for posts, spread for wheels
- **vs. Man:** Bunch/stack for rubs
- **vs. Zone:** Spread for spacing

**Formation Stem & Shifts:**
- Start in one formation, shift to another
- **Purpose:**
  - Force defensive adjustments
  - Get to better formation
  - Create communication issues
  - Catch defense in wrong personnel
