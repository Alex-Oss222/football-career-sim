## Game-Day clock management

### Clock Management Situations

**Under 2 Minutes - First Half:**
- **With Timeouts:**
  - Can use sidelines to stop clock
  - Can run and call timeout
  - Normal play-calling with clock awareness
  - Take shot before half if in range
  
- **Without Timeouts:**
  - Must use incomplete passes or sideline
  - Spike if needed to stop clock
  - Quick-hitting pass plays
  - Don't run unless certain to score

**Under 2 Minutes - Second Half (Trailing):**
- **With Timeouts:**
  - Balance between moving chains and clock
  - Use timeouts strategically
  - Can run if getting yards
  - Mix in passes with runs
  
- **Without Timeouts:**
  - Must throw or use sideline
  - Hurry-up offense
  - Spike when needed
  - Quick passes only

**Under 2 Minutes - Second Half (Leading):**
- **With Timeouts (Defense):**
  - Run to force their timeouts
  - First downs end game
  - Simple, high-percentage plays
  - Don't fumble
  
- **Without Timeouts (Defense):**
  - Kneel if have first down
  - If don't have first down, run clock to <40 seconds then run play
  - Safe plays only
  - Protect lead

## Game-Day two-minute operation

### Two-Minute Offense Operation

**Structure:**
- Pre-designed package of plays
- Everyone knows plays without huddle
- Communication through signals
- Fast tempo execution

**Play Selection in Two-Minute:**
- **Throws that stop clock:** Incomp, out of bounds, spikes
- **Throws that keep clock running:** Middle of field catches
- **Clock stoppers:** Timeouts, first downs, end of quarter

**Two-Minute Approach:**

**With Timeouts:**
- Can run or pass
- Use sideline to stop clock
- Can use middle of field
- Save timeouts for end

**Without Timeouts:**
- Must throw incomplete or OB
- Spike to stop clock if needed
- No runs unless certain score
- Quick-hitting passes only

**Play Types in Two-Minute:**
- **Spikes:** Stop clock
- **Sideline throws:** Outs, corners, fades
- **Quick game:** Get yards and OB
- **Intermediate:** Can use if time
- **Deep shots:** If time and situation right

**Situation Management:**
- **Own territory:** Move quickly but don't force
- **Midfield:** Balance speed with possession
- **Scoring range:** Attack end zone
- **Clock under 30 seconds:** Every second matters
