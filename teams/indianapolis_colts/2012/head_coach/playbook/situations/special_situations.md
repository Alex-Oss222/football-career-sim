## Game-Day field position

### Field Position Considerations

**Own 1-10 Yard Line (Backed Up):**
- **Primary Goal:** Get breathing room, avoid safety
- **Run:** Inside zone, power schemes - get positive yards
- **Pass:** Quick game only if defense loads box
- **Avoid:** Sacks, negative plays, holding penalties
- **Personnel:** 12 or 21 for extra protection
- **No:** Deep shots, complex routes, risky plays

**Own 11-30 Yard Line (Deep Territory):**
- **Primary Goal:** Move chains, don't give short field
- **Run:** Balanced run game, establish physicality
- **Pass:** Quick game, some intermediate routes
- **Approach:** Conservative but not predictable
- **Shot Plays:** Only if very favorable matchup
- **Risk Tolerance:** Low - don't hurt field position

**Own 31-50 Yard Line (Between Barriers):**
- **Primary Goal:** Move into scoring position
- **Run:** Full complement of run game
- **Pass:** All pass game available
- **Approach:** Balanced, trust game plan
- **Shot Plays:** Available on favorable downs
- **Risk Tolerance:** Medium - calculated aggression

**Opponent 49-35 Yard Line (Fringe FG):**
- **Primary Goal:** Get into clear FG range
- **Run:** Run game still viable
- **Pass:** Intermediate routes, calculated shots
- **Approach:** Aggressive but smart
- **Shot Plays:** Yes, but have checkdowns
- **Risk Tolerance:** Medium-High - score is priority
- **4th Down:** Consider going for it if 4th & short

**Opponent 34-21 Yard Line (Clear FG Range):**
- **Primary Goal:** Score TD, settle for FG if needed
- **Run:** All runs available, mix in PA
- **Pass:** Full passing tree
- **Approach:** Aggressive, trust concepts
- **Shot Plays:** Yes, especially PA shots
- **Risk Tolerance:** Medium-High - have FG safety net
- **4th Down:** Attempt FG unless 4th & very short

**Opponent 20-11 Yard Line (Red Zone):**
- **Primary Goal:** Score TD
- **Run:** Power concepts, goal line looks
- **Pass:** Condensed spacing routes, fades, corners
- **Approach:** Physical, attack end zone
- **Personnel:** Increase 12/13/22 usage
- **Risk Tolerance:** Medium - need points
- **4th Down:** Aggressive on 4th & short

**Opponent 10-1 Yard Line (Goal Line):**
- **Primary Goal:** Score TD
- **Run:** Power, iso, QB sneak, wham
- **Pass:** Fades, slants, spot routes, rubs
- **Approach:** Very physical, execution critical
- **Personnel:** Heavy packages (13/22/23)
- **Risk Tolerance:** High - must score
- **4th Down:** Go for it on 4th & goal from 2 or less

## Game-Day hash / field location

### Hash/Field Location

**Left Hash:**
- Field (right side): More space, easier for right-handed QB
- Boundary (left side): Compressed, harder to work
- **Tendencies:** Attack field side more, use boundary for constraint
- **Formations:** Trips/3x1 to field common

**Middle of Field:**
- Balanced space both sides
- No inherent advantage either direction
- **Tendencies:** Can attack either side equally
- **Formations:** 2x2 balanced formations

**Right Hash:**
- Field (left side): More space
- Boundary (right side): Compressed
- **Tendencies:** Attack field side more
- **Formations:** Trips/3x1 to field common

**Strategic Use:**
- Run to field side: More space for cutback
- Pass to field side: More route combinations
- Screens to field side: More room to work
- Use boundary for constraint plays and power runs

## Game-Day score considerations

### Score Considerations

**Leading by 21+ Points:**
- Run clock, limit possessions
- Increase 21/22/13 personnel
- Run-heavy (70%+ runs)
- Simple, physical plays
- No risky shots
- Protect lead, avoid turnovers

**Leading by 14-20 Points:**
- Run clock but stay aggressive enough
- Increase run-pass balance to 60-40 run
- Continue attacking but safely
- Use play-action off run game
- Calculated shots on good down-distance
- Don't let them back in game

**Leading by 7-13 Points:**
- Standard approach, don't get conservative
- Normal personnel distribution
- Balanced run-pass (50-50)
- Attack documented weaknesses
- Take shots when appropriate
- Keep scoring, don't just run clock

**Leading by 1-6 Points:**
- Play to extend lead
- Balanced approach
- Attack aggressively
- Trust concepts that worked
- Take calculated shots
- Don't play scared/conservative

**Tied Game:**
- Standard game plan
- Trust preparation
- Attack weaknesses identified
- Balanced approach
- Make explosive plays
- Execute fundamentals

**Trailing by 1-6 Points:**
- Slight urgency but not panic
- Can still run effectively
- Balanced approach
- May use tempo in 4th quarter
- Attack documented weaknesses
- Execute to take lead

**Trailing by 7-13 Points:**
- Need two scores potentially
- Increase pass rate (60-40 pass)
- Some tempo in 4th quarter
- Aggressive on 4th down
- Attack constantly
- Keep clock in mind

**Trailing by 14-20 Points:**
- Need multiple scores
- Pass-heavy approach (65%+ pass)
- Tempo when appropriate
- Very aggressive on 4th down
- Take more shots
- Manage clock for possessions

**Trailing by 21+ Points:**
- Desperate situation
- Pass-heavy (75%+ pass)
- Constant tempo
- Go for it on 4th frequently
- Take risks
- Score quickly

## Game-Day special situation protocols

### Special Situation Protocols

**End of Half Situations:**
- Assess time and timeouts
- Determine if trying to score or running clock
- Execute two-minute if scoring
- Take knee if running clock

**Victory Formation:**
- Leading, under 2 minutes, defense out of timeouts
- Line up in victory formation
- Take knee repeatedly
- Game over

**Prevent Quick Score (Defense has ball, little time):**
- Not applicable for offense

**Coming Out After Halftime:**
- Execute adjustments from halftime
- Attack their halftime adjustments
- Show new wrinkles prepared
- Set tone for second half

**Weather Considerations:**

**Heavy Rain:**
- More run-heavy
- Simpler passes
- Ball security critical
- Handoffs more reliable

**High Wind:**
- More run-heavy
- Avoid deep passes into wind
- Attack with wind behind
- Throw low trajectory passes

**Extreme Cold:**
- Ball harder to grip
- Focus on execution
- Simpler is better
- Ball security paramount

**Heat/Humidity:**
- Rotate players more
- Use tempo to tire defense
- Hydration critical
- Second half advantage if conditioned

## Game-Day game flow management

### Game Flow Management

**Possession Philosophy:**

**Short Fields (backed up):**
- Move chains, establish rhythm
- Don't hurt field position
- Physical runs early

**Long Fields (good position):**
- Take more risks
- Attack documented weaknesses
- Balance efficiency with explosiveness

**Scoring Opportunities:**
- Red zone execution critical
- Take points when available
- Don't settle for FG too early
- Attack end zone when able

**Responding to Defensive Adjustments:**

**When Defense Adjusts:**
1. Identify the adjustment
2. Have counter-punch ready
3. Attack adjustment immediately
4. Force them to adjust again

**Common Defensive Adjustments:**
- More pressure → screens, quick game
- Different coverage → re-identify with motion
- Bracket best WR → attack elsewhere
- Load box → pass game
- Play soft → intermediate routes
- Change fronts → run to new bubble

**When Offense Stalls:**
1. Review what's working vs. not
2. Re-establish successful concepts
3. Try new formation/personnel combo
4. Take calculated shot to loosen defense
5. Use tempo to catch defense off guard

**Building Off Success:**
- When something works, have follow-ups ready
- **Example:** Run works → play-action off same look
- **Example:** Slant works → slant-and-go next time
- **Example:** Defense adjusts → constraint play ready

**Complementary Concepts:**
- Every successful play should set up another
- **Run game complements:**
  - Outside zone → Counter
  - Inside zone → Boot action
  - Power → Play-action shot
  
- **Pass game complements:**
  - Quick game → Shot off same look
  - Intermediate → Deep off same concept
  - Screen → Run after screen look
