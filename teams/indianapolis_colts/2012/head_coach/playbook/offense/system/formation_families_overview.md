## Game-Day System Formation Inventory Integration (exact extract)

### Formation Inventory Integration

**11 Personnel Formation Families:**
- Trips variations (20): Standard, Open, Closed, Bunch, Stack, Nasty, Wing, Over, Slot, Wide, Tight, Flex, Y-Off, Trey, Diamond, Orbit, Doubles, Ace, Offset, Split
- 2x2 Spread (20): Standard, Tight, Wide, Stack, Bunch, Flex, Twin, Pro, Pistol, Empty, Doubles, Orbit, Nasty, Offset, Y-Stick, Ace, Slot, Wing, Over, Trey
- 3x1 Sets (20): Open, Closed, Nub, Detached, Strong, Weak, Stack, Bunch, Diamond, Wide, Trey, Slot, Wing, Y-Off, Nasty, Orbit, Doubles, Ace, Offset, Split
- Bunch/Stack (20): variations with different alignments and spacing
- Pistol (20): variations with backfield positioning changes
- Empty (20): Quads, Trips, Spread, Trey, Diamond, Bunch, Stack, plus variations
- Gun Spread (20): Base with multiple sub-variations

**12 Personnel Formation Families:**
- Pro Sets (20): Right/Left, Strong, Weak, Wing, Slot, Twin, Split, plus variations
- Gun Strong (20): Right/Left, Twins, Wing, Slot, Close, Split, plus variations
- Pistol (20): Strong, Weak, Ace, Twins, Wing, Slot, plus variations
- I-Formation/Singleback (20): Pro, Strong, Weak, Twins, Slot, Wing, plus variations

**21 Personnel Formation Families (Dual-Back Identity):**
- Pony (20): Right/Left, Strong, Weak, Twins, Stack, Split, plus variations
- Split Back (20): Right/Left, Twins, Stack, Wing, Slot, plus variations
- I-Formation (20): Pro, Strong, Weak, Twins, Wing, Slot, plus variations

**Weekly Formation Selection:**
- Select 15-20 formations per personnel group for game plan
- Prioritize formations that create best matchups vs. opponent
- Ensure formations allow for multiple play types (run, pass, RPO, screen)
- Build formation families that allow easy adjustments via motion/shifts
